# Social Media Application Using Graph Database

## Overview

This repository contains my M.Tech Advanced Database Management System (ADBMS) mini project.

The project demonstrates the development of a real-time social chat application using a Graph Database (Neo4j). Unlike traditional relational databases, Neo4j models users, messages, and relationships as interconnected nodes, enabling efficient traversal and relationship-based queries.

The application provides user authentication, real-time messaging, group interactions, and graph-based data management through a modern full-stack architecture.

## Objective

The primary objective of this project is to explore Graph Database concepts by developing a social media chat application that efficiently manages user relationships and messaging using Neo4j.

## Graph Database Design

The application models:

* User Nodes
* Message Nodes
* SENT Relationships
* RECEIVED Relationships

This graph-based approach enables efficient querying of social connections and messaging relationships.
